Primer

Primer is a minimal mobile-first CSS boilerplate that provides helpful styling for buttons, forms, tables, lists, and typography for quick prototyping.

Download and Installation

Primer uses CSS variables to configure styles. normalize.css is used acheive browser consistent styles.

It also uses precss and autoprefixer, PostCSS plugins to create support custom-media queries, custom-selectors and fallbacks to CSS variables.

  1. Clone the repo.
  2. Run npm install to install dev dependencies for build.
  3. Modify variables values in the file _variables.css
  4. Run gulp build to generate the css file.
  5. Generated CSS file is found in the dist folder.

HTML

A basic HTML5 template that does the following:

  • Sets the language of the content. Default to en
  • Defines the viewport meta tag to enable responsiveness and consistency.
  • Includes link to primer CSS file
  • Includes link to a blank JS file
<!doctype html>
<html lang="en">

<head>
  <meta charset="utf-8">
  <meta http-equiv="x-ua-compatible" content="ie=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1,shrink-to-fit=no">

  <title></title>
	<meta name="description" content=" ">
  <link rel="stylesheet" href="css/main.min.css">
</head>

<body>

<!-- Add your content here -->

<script src="js/script.js"></script>
</body>

</html>

Grids

Grids are intentionally left out. Using CSS Grid to build custom layouts better and faster. More details about how to build layouts using CSS Grids can be found at MDN

Typography

The font size is set to 62.5% on the html element, so all font sizing will cascade from the root.

So, 1rem = 10px instead of 16px which is default for a browser. To set any pixel value to rem, we can just divide it by 10.

Lets say you want to set a value of a <p> tag to 18px. We can set it as 1.8rem

Headings all have the same line-height and margins, and only the font size is individually defined.

Blockquote

Blockquote is set to have a left border, with the browser default padding removed. A cite will be right aligned and italicized. 

<blockquote>
	<p>If you tell the truth, you don't have to remember anything.</p>
	<cite>― Mark Twain</cite>
</blockquote>

Screen Reader Text

Accessibility is important - if you have a label that should be hidden but accessible, or use an icon as a link, or want to skip navigation, use the screen-reader-text class to hide content visually. 

<a class="screen-reader-text" href="#content">Skip Navigation</a>

Responsive Images

Add a responsive-image class for responsive images.

<img src="#" class="responsive-image" alt="Responsive Image">
